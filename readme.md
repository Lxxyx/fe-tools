## fe-tools

usage:

```
npm i fe-tools -g
```

    Usage: fe <command>


    Commands:

      babel       set babel, use es2017
      eslint      set eslint
      help [cmd]  display help for [cmd]

    Options:

      -h, --help     output usage information
      -V, --version  output the version number

## fe babel

Auto generate babel config

## fe eslint

Auto generate eslint config
